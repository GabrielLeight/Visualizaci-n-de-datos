- Los datos de "RAWG.io_xbox_games_with_devs.csv" fueron obtenidos por la API de RAWG.io (https://rawg.io/apidocs), una base de datos de videojuegos. 
- Los datos de htlb jsonlines fueron obtenidas por la página Kraggle  https://www.kaggle.com/code/baraazaid/video-game-playtime-analysis/input, donde se realizó un analisis del tiempo de juego de singleplayer, multiplayer, consolas y generos.
el htlb.jsonlines son lineas de input en json que se utilizaron para otro analisis de datos de videojuegos realizado en kraggle,
y esta conformado por varios datos relacionados a los tiempos promedios de los juegos documentados dentro de la página.
